# Gestor de Biblioteca - Sesión 2
Este proyecto implementa la clase Libro con:
- Encapsulación (atributos privados).
- Constructores (por defecto y con parámetros).
- Métodos accessores (Getters y Setters).
Sesión 3: Implementación de la clase Biblioteca utilizando std::vector para gestionar la colección de libros.
